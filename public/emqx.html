EAF MQTT-Client

 Quick Setup

Intro

This Quick setup uses the REST API of a EMQX to setup an EMQX broker to work with the STROMDAO EAF Client.
  • Step 1: Create API key in EMQX for Setup
  • Step 2: Let Setup create  user for the EAF MQTT Client
  • Step 3: Set permissions for topics
