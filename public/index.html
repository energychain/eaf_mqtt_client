Key Function

  • MQTT Broker Connection Utilizes the specified in the .env. file or provided as an environment variable to establish a connection with the MQTT Broker.
  • Topic Subscription Subscribes to the topic stromdao-eaf/metering/updateReading/# , ensuring that it receives all messages published to this topic and its subtopics.
  • Meter Reading Forwarding When a message is received on a topic following the pattern  stromdao-eaf/metering/updateReading/ABC, the module interprets it as a meter reading.
    It then forwards this meter reading to the metering/updateReading microservice method of the STROMDAO EAF.
